Activity Indicator

Activity indicators are a simple element used to show underlying progress within an application. These indicate to the user the application is processing input, and loading output based on a previous request.

A basic activity indicator can be generated by adding the "indicator" class to an empty block level element.

A large variant of the indicator can be generated by using the "indicator-large" class.

The spinner is a smaller version of the activity indicator and comes in three variations, spinner-bounce-bottom, spinner-bounce-middle, and spinner-bounce-top which are displayed below. Unlike the previous examples, the spinner can be used on inline elements such as spans.
