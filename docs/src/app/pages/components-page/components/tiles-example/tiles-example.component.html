Tiles

Tiles are a navigation component that allow a user to access pages, applications, and other data within a product. At minimum, a tile should possess a recognizable avatar or icon image. Tiles may also be given titles, descriptions, and a favorite icon depending on the size of the tile.

By using the "ias-tile" class, you get the right background color, as well as the border color and cursor type when you hover over the tile. The styling of the elements within the tile is up to you.

Tiles can be laid out in a wrapping grid using the CSS Grid layout. See the definition for the "app-grid" class for an example.
