Tags

Tags are a prevalent component used throughout applications. They can be found tied to search inputs as filters, within email messages as recipients, at the bottom of a product page as item identifiers and many other places.

The Combobox element is a mixture of different input components rolled into one. It holds and displays tags, has an input form field, and allows for a drop-down element to see predetermined options.

The combobox offers a variety properties to be set for customization. Use the "Customize Example" accordion to see the effects of different values.

Dropdown menus have a similar appearance to a combobox, but their options are not as flexible. Dropdowns only allow for a user to add tags from a predetermined list of values revealed by clicking on the element.
