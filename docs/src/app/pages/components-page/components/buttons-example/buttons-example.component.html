Buttons

IAS buttons are implemented by changing the theme for Bootstrap's standard button classes, such as: "btn", "btn-primary", and "btn-outline-primary". You can simply use Bootstrap buttons as you normally would, as you can see from the examples below. Note: the IAS theme doesn't define buttons for for Bootstrap's secondary, success, danger, warning, info, light, or dark predefined styles. So no changes have been made for them.
